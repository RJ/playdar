Windows

Playdar Installer

Built with NSIS by lozzd
Provides an easy installation and setup. Install to Program Files directory, and then provides a GUI to edit the config file, and search music directories.
Features:
  • Wraps playdar.exe and scanner.exe completely, and shows log output.
  • Minimises to tray
  • Colour coding of important events in log
  • Provides Balloon notifications for events
You can choose not to use the wrapper, and just use the command line tools if you wish.
